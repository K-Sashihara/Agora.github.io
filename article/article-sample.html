理論背景Theoretical Background

相互作用するスピン系のハミルトニアンは以下のように記述されます：

The Hamiltonian of the interacting spin system is described as follows:

\[ H = -J \sum_{\langle i,j \rangle} s_i s_j - h \sum_{i} s_i \]

ここで、\(J > 0\) は強磁性相互作用、\(s_i \in \{+1, -1\}\) は各格子点上のスピンを表します。

Where \(J > 0\) represents the ferromagnetic interaction and \(s_i \in \{+1, -1\}\) represents the spin at each lattice site.

The core part calculating the spin-flip probability is as follows:

def update_spins(lattice, beta, J):
    N = len(lattice)
    for _ in range(N*N):
        x, y = np.random.randint(0, N, 2)
        dE = 2 * J * lattice[x, y] * (
            lattice[(x+1)%N, y] + lattice[(x-1)%N, y] +
            lattice[x, (y+1)%N] + lattice[x, (y-1)%N]
        )
        if dE <= 0 or np.random.rand() < np.exp(-beta * dE):
            lattice[x, y] *= -1
